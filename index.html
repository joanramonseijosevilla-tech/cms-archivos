Sistema Publicaciones Panel

Web corporativa editable sin servidor

Mini CMS para publicar contenido sin tocar código.

Una solución estática, económica y fácil de mantener: el cliente publica desde un panel privado, Make actualiza GitHub y la web se genera automáticamente desde un archivo JSON.

Ver publicaciones Cómo funciona
Arquitectura GitHub Pages + JSON + Make

Frontend estático, backend automatizado y token de GitHub protegido fuera del navegador.

Sistema modular

Diseñado para clientes no técnicos

El cliente solo rellena un formulario. Make se encarga de subir imágenes, actualizar el JSON y dejar GitHub Pages listo para servir el contenido.
