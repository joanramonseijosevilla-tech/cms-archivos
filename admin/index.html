Ayuda

Guía rápida del panel

Consulta los pasos básicos para crear, organizar y recuperar publicaciones.

Crear una publicación
  1. Escribe el título.
  2. Añade la descripción usando el editor.
  3. Elige categoría y estado.
  4. Selecciona una imagen y completa el texto alternativo.
  5. Pulsa Guardar publicación.
Editar o duplicar

Usa Editar para modificar una publicación existente. Puedes cambiar textos, categoría, estado o imagen.

Usa Duplicar para reutilizar una publicación como base. La copia se abre en el formulario y queda oculta hasta que decidas publicarla.

Publicar, ocultar y papelera

Publicado aparece en la web pública. Oculto se conserva en el panel, pero no se muestra en la web.

Papelera sirve para retirar una publicación antes de eliminarla definitivamente.

Ordenar publicaciones

Usa Ordenar por sección para reorganizar todas las publicaciones o solo una categoría.

Dentro del modo ordenar puedes mover una publicación una posición, enviarla al principio o al final, y después guardar o cancelar los cambios.

Búsqueda, filtros y paginación

La búsqueda localiza publicaciones por título, descripción o categoría.

Los filtros permiten ver publicaciones por estado o categoría. La paginación permite revisar el listado por bloques de 10, 25 o 50.

Acciones por lote

Marca varias publicaciones para publicar/ocultar, mover a papelera o cambiar categoría en una sola acción.

Si hay filtros activos, Seleccionar mostradas selecciona solo las publicaciones visibles en ese momento.

Copias y recuperación

Descargar copia completa guarda un ZIP con los datos y las imágenes usadas.

Verificar copia comprueba si el ZIP parece correcto.

Recuperar publicaciones eliminadas busca publicaciones que estaban en la copia y ya no aparecen en el panel.

Imágenes

Al subir una imagen, el panel la optimiza automáticamente para que pese menos y mantenga su proporción.

El texto alternativo ayuda a describir la imagen y mejora la accesibilidad.

Nueva publicación

Sube una imagen, título y descripción. Make guardará el cambio en GitHub cuando esté conectado.

Descripción

Formato básico: títulos, negrita, cursiva, colores controlados, viñetas, lista numerada, sangría y alineación. Al pegar desde otro editor se conserva el formato razonable y se limpia lo no seguro.

Herramientas

Herramientas del CMS

Opciones de mantenimiento y seguridad para trabajar con el contenido del panel.

Descargar copia completa

Genera un ZIP con los datos y las imágenes usadas por las publicaciones actuales.

Úsala antes de cambios importantes para conservar una copia del contenido.

Verificar copia completa

Selecciona un ZIP descargado desde este panel para comprobar si contiene los datos y las imágenes esperadas.

Solo revisa la copia seleccionada. No realiza cambios.

Recuperar publicaciones eliminadas

Selecciona una copia completa para buscar publicaciones que ya no están en el panel y recuperarlas.

Añade solo las publicaciones que elijas. Las publicaciones actuales se conservan.
