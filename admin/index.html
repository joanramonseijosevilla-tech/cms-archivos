Manual del panel

Guía estable para crear, editar, ordenar, proteger y recuperar publicaciones desde el mini CMS.

Versión 9.1 Manual actualizado con categorías, papelera, lotes, copias, recuperación, vista compacta, orden por categoría, imágenes más claras y límite mensual del panel.
Idea clave Revisa los cambios en pantalla y usa los botones de guardar o confirmar solo cuando quieras aplicarlos.
1. Crear una publicación
  1. Rellena el título.
  2. Escribe la descripción con el editor.
  3. Elige la categoría: Galería, Proyectos o Novedades.
  4. Elige si empieza como publicada u oculta.
  5. Selecciona una imagen y añade el texto alternativo.
  6. Pulsa Guardar publicación.

Si falta algún dato obligatorio, el panel te avisará antes de guardar.

2. Imagen de la publicación

La zona de imagen indica si hay una imagen pendiente, una imagen actual, una imagen reutilizada o una nueva imagen seleccionada.

  1. Al crear, debes elegir una imagen.
  2. Al editar, puedes mantener la imagen actual sin tocar nada.
  3. Si eliges una nueva imagen, se aplicará al guardar.
  4. Si te equivocas, usa Quitar imagen elegida antes de guardar.

El panel optimiza la imagen automáticamente para que pese menos y mantenga su proporción.

3. Editor de descripción

El editor permite formato básico: títulos, negrita, cursiva, listas, sangría, alineación y colores controlados.

Al pegar texto desde otro editor, el panel intenta conservar un formato razonable y limpiar elementos innecesarios.

4. Editar, duplicar o cancelar

Editar carga la publicación en el formulario para modificar textos, categoría, estado o imagen.

Duplicar crea una copia como borrador oculto para reutilizar una publicación anterior sin cambiar la original.

Si empiezas a cambiar algo y sales, cancelas o intentas editar otra publicación, el panel te avisa si hay cambios sin guardar.

5. Publicado, oculto y papelera

Publicado aparece en la web pública. Oculto se conserva en el panel, pero no se muestra en la web.

Mover a papelera retira la publicación sin eliminarla del todo.

Dentro de la papelera puedes restaurar o eliminar definitivamente.

6. Categorías

Las publicaciones se organizan en Galería, Proyectos y Novedades.

Las categorías sirven para filtrar el panel y para ordenar mejor lo que después se verá en cada sección de la web pública.

7. Ordenar publicaciones

Los botones Subir y Bajar sirven para ajustes rápidos de orden.

Usa Ordenar por categoría cuando quieras ordenar solo Galería, solo Proyectos o solo Novedades sin ver todo mezclado.

  1. Entra en Ordenar por categoría.
  2. Elige Todas, Galería, Proyectos o Novedades.
  3. Mueve las publicaciones con Primera, Subir, Bajar o Última.
  4. Pulsa Guardar orden o Cancelar cambios.
8. Búsqueda, filtros y contadores

La búsqueda localiza publicaciones por título, descripción o categoría.

Los filtros permiten ver solo publicadas, ocultas, papelera o una categoría concreta.

Los contadores ayudan a saber cuántas publicaciones hay en total, cuántas están publicadas, ocultas o visibles en ese momento.

9. Paginación y vistas

La paginación permite revisar el listado por bloques de 10, 25 o 50 publicaciones.

Vista completa muestra más información. Vista compacta reduce el listado para revisar muchas publicaciones más rápido.

Ver más / Ver menos permite desplegar o recoger descripciones largas.

10. Acciones por lote

Marca varias publicaciones para aplicar una acción común: publicar, ocultar, mover a papelera, restaurar, eliminar definitivamente o cambiar categoría.

Si hay filtros activos, Seleccionar mostradas selecciona solo las publicaciones visibles en ese momento.

Para cambiar categoría en bloque, puedes asignar una misma categoría a todas o revisar una por una antes de guardar.

11. Copias y recuperación

Descargar copia completa genera un ZIP con los datos y las imágenes usadas.

Verificar copia revisa si el ZIP seleccionado parece correcto.

Recuperar publicaciones eliminadas compara una copia con el panel actual y permite recuperar publicaciones que ya no están.

La recuperación está pensada para publicaciones eliminadas definitivamente; si una publicación sigue en la papelera, todavía no aparecerá como recuperable.

12. Uso en móvil

En móvil tienes un menú rápido para saltar a formulario, publicaciones, filtros, lotes, ordenar, vista, herramientas y manual.

La barra inferior permite moverse por las zonas principales sin tener que hacer mucho desplazamiento.

13. Uso mensual del panel

En Herramientas puedes ver el uso mensual del panel. Es un límite interno para controlar las acciones que guardan cambios.

Buscar, filtrar, cambiar de página, abrir el manual, verificar copias o cambiar de vista no suma uso.

Si se alcanza el límite mensual, el panel avisa antes de guardar y no realiza el cambio.

14. Buenas prácticas
  1. Antes de cambios importantes, descarga una copia completa.
  2. Usa Oculto para preparar publicaciones antes de enseñarlas en la web.
  3. Usa categorías y orden por categoría para mantener la web pública más clara.
  4. Escribe textos alternativos sencillos para las imágenes.
  5. Después de subir una versión nueva del panel, abre la URL con el número de versión actualizado.
